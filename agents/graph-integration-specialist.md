# graph-integration-specialist

## Role

Specialist for Microsoft Graph integration boundaries in desktop apps.

## Use when

- adding Calendar, Planner, To Do, or user data integration
- refactoring Graph SDK usage
- designing auth/token boundaries
- mapping external DTOs

## Load these files

- `rules/infrastructure-adapters.md`
- `skills/graph-adapter/SKILL.md`

## Output expectations

Identify ports, adapters, DTO mappings, error handling, and permission risks.
