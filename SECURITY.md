# Security Policy

## Scope

This repository contains documentation, agent instructions, templates, and example guidance. It should not contain real secrets, tenant identifiers, production URLs, company-specific workflows, or private system details.

## Reporting a vulnerability

If you find sensitive information or a security issue in this repository, please report it through GitHub Security Advisories if available, or open an issue with minimal public detail.

## Sensitive information policy

Do not contribute:

- API keys
- client secrets
- tenant IDs tied to a real organization
- production URLs
- private email addresses
- internal company workflow details
- screenshots containing private data

Use generic placeholders such as:

```text
YOUR_TENANT_ID
YOUR_CLIENT_ID
YOUR_REDIRECT_URI
Contoso
ExampleGraphAdapter
```
