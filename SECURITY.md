# Security Policy

## Scope

This repository contains documentation, agent instructions, templates, and example guidance. It should not contain real secrets, tenant identifiers, production URLs, company-specific workflows, private system details, or private Microsoft 365 organization data.

## Reporting a vulnerability

If you find sensitive information or a security issue in this repository, use GitHub Security Advisories if available, or open an issue with minimal public detail. Do not paste secrets into public issues.

## Sensitive information policy

Do not contribute API keys, client secrets, tenant IDs tied to a real organization, production URLs, private email addresses, internal company workflow details, screenshots containing private data, or real M365 organization names, IDs, or URLs.

Use generic placeholders such as `YOUR_TENANT_ID`, `YOUR_CLIENT_ID`, `YOUR_REDIRECT_URI`, `ExampleApp`, `ExampleGraphAdapter`, and `example.invalid`.
