# Code of Conduct

This project aims to be a practical and respectful space for improving AI-assisted .NET desktop development.

## Expected behavior

- Be respectful and constructive.
- Focus criticism on ideas and implementation details.
- Prefer examples and actionable suggestions.
- Assume good intent.

## Unacceptable behavior

- Harassment or personal attacks
- Publishing private or sensitive information
- Intentionally misleading guidance
- Repeated off-topic disruption

## Reporting

Open a GitHub issue or contact the maintainer through the repository owner profile if you need to report a concern.
