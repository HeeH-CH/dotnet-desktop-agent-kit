# Code of Conduct

This project aims to be a practical and respectful space for improving AI-assisted .NET desktop development.

## Expected behavior

- Be respectful and constructive.
- Focus criticism on ideas, examples, and implementation details.
- Prefer actionable suggestions.
- Assume good intent.
- Protect private information.

## Unacceptable behavior

- Harassment or personal attacks.
- Publishing private or sensitive information.
- Intentionally misleading guidance.
- Repeated off-topic disruption.
- Submitting examples that contain real secrets or private organization details.

## Reporting

Open a GitHub issue or contact the maintainer through the repository owner profile if you need to report a concern. Do not include sensitive details in a public issue.
